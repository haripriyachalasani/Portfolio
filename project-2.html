Project Overview

Synapse Hub is an e-learning website designed and developed using HTML, CSS, and JavaScript. It Aimed to create a user-friendly platform to help mentees connect with mentors easily and upskill themselves. The Synapse Hub website was mobile-responsive, providing seamless accessibility across various devices.

Synapse Hub has many features as it facilitates smooth mentor-mentee interactions and knowledge sharing. It has received positive user feedback on the website's usability and effectiveness in achieving the project's goals.
